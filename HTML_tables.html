Table Attributes

Align: It aligns the table. (left, right and center)
bgcolor: Sets the background color of the table. Colors are defined in hex value codes.
border: Sets the border around table. It is defined in pixels.
cellspacing: Defines the space between cells.
cellpadding: Defines the padding in the table cell.
frame: Defines which side of the frame must be displayed.
width: Defines the width of The table.
